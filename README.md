# ProPlanner — Система за управление на AI ресурси

ProPlanner е модерна Full-Stack платформа за съхранение, управление и оценяване на екипни ресурси и AI инструменти.

## 🚀 Технологичен стак
- **Backend:** Laravel 11, MySQL, Sanctum, Spatie Permissions
- **Frontend:** Next.js (React), Tailwind CSS, Axios
- **Среда:** Docker (Laravel Sail)

---

## ⚙️ Инструкции за инсталация и стартиране (Docker)

### 1. Бекенд (Laravel)
1. Отворете терминал в папка `backend`.
2. Инсталирайте зависимостите: `composer install`
3. Настройте средата: `cp .env.example .env` и `php artisan key:generate`
4. Стартирайте Docker: `./vendor/bin/sail up -d`
5. Изпълнете миграциите: `./vendor/bin/sail artisan migrate`
6. Инициализирайте категориите през браузъра: `http://localhost:8000/api/force-categories`

### 2. Фронтенд (Next.js)
1. Отворете терминал в папка `frontend`.
2. Инсталирайте пакетите: `npm install`
3. Стартирайте проекта: `npm run dev`
4. Отворете `http://localhost:3000`

## 🔐 Ролева система (ACL)
- **Owner:** Админ достъп. Одобрява задачи. Директно публикуване.
- **User:** Разглежда ресурси. Добавя задачи (чакащи одобрение). Оценява инструменти.

---

## 🛠️ Добавяне и Оценяване на инструменти
- **Добавяне:** През Дашборда -> "Добави задача". Попълват се детайли и картинка.
- **Рейтинг (Бонус):** Всеки одобрен инструмент може да получи 1-5 звезди и текстово ревю.
